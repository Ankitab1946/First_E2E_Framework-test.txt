User --> SIDE
SIDE --> DD
SIDE --> AUDIT_UI
SIDE --> PROMPT_UI

DD --> FILTER_API : Apply filters
FILTER_API --> DICT_SVC
DICT_SVC --> DB : Fetch active records
DB --> DD : Show filtered grid

DD --> POPUP : Add / View / Edit record
POPUP --> CRUD_API : Save changes
CRUD_API --> FINAL_SVC
FINAL_SVC --> DB : Insert / Update / Soft Delete / Reactivate
FINAL_SVC --> DB : Write audit + history
DB --> DD : Refresh grid

DD --> S3_SVC : Save Extracts to S3
S3_SVC --> DB : Read latest data from 4 tables
S3_SVC --> S3 : Upload timestamped CSV files

AUDIT_UI --> AUDIT_API : Search audit history
AUDIT_API --> DB : Fetch audit records
DB --> AUDIT_UI : Show audit grid

PROMPT_UI --> DB : Read prompts
DB --> PROMPT_UI : Show read-only prompts

note bottom of DD
Grid refresh happens after:
filter change, add, edit,
soft delete, reactivate, or manual refresh.
end note
